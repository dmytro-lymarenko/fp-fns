### [lib](../../)/[common](../)/original

#### Declaration
```js
function original([arg]) {...}
```

Returns the same arg as it is given.

#### Since
1.0.0

#### Arguments

* **arg (any)**: The argument which will be returned.

#### Returns

**(any)**: Returns the given **arg**.

#### Example

```js
original(5)
// => 5

original({ foo: 'bar' })
// => { foo: 'bar' }

```
